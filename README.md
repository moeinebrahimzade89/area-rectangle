# 📐 Rectangle Area Calculator

A simple Python command-line program that calculates the area of a rectangle by taking the **length** and **width** as input from the user.

## 🛠 Technologies Used
- Python 3.x

## ✨ Features
- Accepts user input for **length** and **width**
- Calculates the rectangle area using the formula:

  \[
  \text{Area} = \text{Length} \times \text{Width}
  \]

- Handles invalid input with `try-except`
- Displays a clear error message if the entered value is not a number

## 🚀 How to Run
1. Save the file as `area rectangle.py`
2. Open your terminal or command prompt
3. Run the program with:
```bash
python "area rectangle.py"
